등급 기준

별표는 해당 언어에 대한 숙련도를 나타냅니다.
0에서 6까지 60등분하여 숙련도를 평가하며, 숫자가 높을수록 높은 숙련도를 의미합니다.

  • ⭐0.0 : 경험이 아예 없는 경우
  • ⭐1.0 : 배우고 활용해본 적은 있지만, 실제 프로젝트를 진행해보지는 않음
  • ⭐2.0 : 프로젝트를 진행해본 경험이 있음
  • ⭐3.0 : 개인 프로젝트에서 쉽게 활용할 수 있음
  • ⭐4.0 : 공동 프로젝트를 무리 없이 진행할 수 있음
  • ⭐5.0 : 실무에서 주력으로 사용할 수 있음
  • ⭐6.0 : 전문가 수준으로 다룰 수 있음
