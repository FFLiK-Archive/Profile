한글, 난해한 프로그래밍 언어 제작 프로젝트

한글이라는 언어는 난해한 프로그래밍 언어로, 우리가 채팅을 할 때 자음과 모음으로 대화하는 것에서 착안하여, 마치 대화하듯이 프로그래밍을 할 수 있는 언어입니다.
입력으로 들어온 코드를 인코더 객체의 문자열 분석을 통해 토큰 단위로 나누고, 이를 인터프리터 객체로 한 줄씩 읽어들여 실행하는 방식으로 동작합니다. 이진법 기반의 내부 데이터 관리 시스템을 통해 변수와 배열이 관리되며, 토큰 기반으로 이루어진 문장을 분석하여, 대입인지, Goto인지, 함수 호출인지 등을 판단하고, 식이 포함되어있다면, 연산자 우선순위에 따라 후위 표기법으로 변환해 계산하여 처리합니다. 함수 호출은 HFDL(Hangul Function Development Library)를 통해 제작된 프로그램을 실행하는 방식으로 수행되며, 표준 입출력으로 매개변수를 전달하고, 반환값을 전달받는 시스템을 활용합니다.
이 프로젝트는 2022년도 한국과학영재학교 R&E 프로그램 중 하나로 수행되었습니다.
